# Features

## Modularity

With a single codebase LibreMesh do a `reasonable-defaults` choice for most common mesh protocols: `Babel`, `B.A.T.M.A.N. advanced`, `BMX7`, `OLSR`.

It could be used to join to an existing mesh network with minimal configuration.

It could be used as a testbed for different protocols configuration.
