
# Monitoring

Monitoring of LibreMesh nodes could be done in several ways.

Some example from communities using LibreMesh

## [grafana.altermundi.net](https://grafana.altermundi.net)
Uses `Victoria Metrics` and `Grafana`

## [panorama.antennine.org](https://panorama.antennine.org)
Uses `Prometheus` and `Grafana`
